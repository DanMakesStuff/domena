Tutorial
Unfortunately, due to the complexity of Mumble mod, the necessary software installation is more demanding unlike the conventional servers.
We have prepared a tutorial for you on how to set everything up.
No worries, in 10 to 20 minutes you will be able to play on the server with others!

And if you have any quiestions or need help, feel free to tell us, we will be more than happy to help!

Files needed:

  • Forge installer 1.16.5 here - Minecraft version that supports mods.
  • Create 1.16.5 here
  • Mumble link 1.16.5 here - Mod for linking Mumble program with minecraft.

  • Mumble here - Standalone program responsible for voice chat.


Forge
After downloading all the necessary files, we'll install minecraft version supporting mods:

  • Forge 1.16.5-36.0.22


Forge needs to be installed to work, after starting the installation, a window should pop up:

Leave Install client selected. Click ok:

Installing Forge will take a second, after it's done, start your preffered minecraft launcher, select Forge 1.16.5 version of minecraft and start the game.


After Minecraft has loaded, the title screen should look something like this:

After this, you can quit the game. We started Minecraft forge for it to create necessary files and documents before we put mods in.
Mumble

Now the most complicated part: Mumble.


First, we start mumble installation, a window should pop up:

Click on Next, we select agreenments and click Next:

A few screens with other options will follow but you dont need to change anything there.



After clicking Finnish Mumble should start:
Setting Mumble correctly

A window for sound testing should show up, you also dont need to change anithing there, skip all of them.

For the voice chat to work properly, we need to head into settings:

Then you have to select Sound output:

And now set all highlited things on the image:
  • Checked Position sound
  • Checked Headphones under Position sound
  • Min distancee - around 1m
  • Bloom - around 150%
  • Max distance - around 30m
  • Max volume - exactly 0%


These settings are based on our experience for the best effect of the proximity chat.
If there's anything you dont understand or something went wrong, feel free to tell us:


Now we move to Addons



Now we need to set Mumble so it links to minecraft:



Here we need to check Link with games and send out position
Without this the voice chat won't work

And we'e done with the worst part, only two steps left:
  • Put downloaded mods into minecraft
  • Add and join Mumbe voice chat server

Mumble server
Now we just need to connect to voice chat (mumble server)

Click the Server button on the top panel, then click Connect... :



A window with servers should pop up, click Add new... here:



Here we fill in server info, your name and server name:
  • Address: mumble-de.cleanvoice.com
  • Port: 39482
  • Username: your minecraft nickname
  • Tag name is just a name of the server that it will be saved under


Now we can test if Mumble is set up properly, start minecraft and join the mumble server, then start a singleplayer world or join a multiplayer server in minecraft. mumble should send a message in the console saying Minecraft linked
