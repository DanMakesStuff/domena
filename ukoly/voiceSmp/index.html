Úvod
Příjde vám, že Minecraft už nenabízí nic nového? Vše začíná být během karantény ohrané a nezáživné?

S tím samým problémem jsme se potýkali i my.
Proto vznikl Voice SMP! První původem Český server s voice chatem inspirovaný populární zahraniční sérií Epic SMP, která nás velmi zaujala a chtěli bychom její unikátní vlastnosti přinést obyčejným hráčům.

Za účelem udržení přátelské a bezstarostné komunity server používá whitelist.

Jestli si u nás chceš zahrát, musíš o tom informovat některého člena našeho týmu (na jakékoli zmíněné sociální síti). Rádi tě uvítáme v naší komunitě, dokud se budeš bavit a zároveň budeš jednat rozumně.

(Jo a taky nemusíš mít zakoupenou verzi hry, aby jsi si u nás zahrál)

Módy
Server zatím má jen dva módy. Časem plánujeme postupně módy přidávat.

  • Create

    Mód Create je propracovaný technický mód, který mimo jíné dokáže pohybovat celými stavbami.

  • Mumble

    Mód Mumble umožňuje hráčům využívat proximity chat, server byl vytvořen pro tento mód.

  • Ostatní módy

    Ostatní módy najdete v sekci Update a na našem Discordu, kde lze stáhnout všechny módy naráz.
Jak se připojit
Vzhledem ke komplexicitě módu Mumble je bohužel instalace potřebného softwaru náročnější než u obvyklých serverů.
Připravili jsme pro vás zde návod, jak vše zprovoznit, do 10 až 20 minut si budete moct zahrát na serveru s ostatními!
